[Android 2.0.1](http://developer.android.com/sdk/android-2.0.1.html "android") is a minor platform release deployable to Android-powered handsets starting in December 2009.

_This release includes minor API changes, bug fixes and framework behavioral changes. For information on changes and fixes, see the Framework API section._
