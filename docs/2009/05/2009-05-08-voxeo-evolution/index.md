[Voxeo's Evolution IVR developer](http://evolution.voxeo.com/docs/evolution_overview.jsp "voxeo") site _gives you two options for XML-based, web integrated IVR development._

_Evolution Community is a completely free, feature filled service that you can use to design, create, and test IVR applications._

_Evolution Professional adds valuable features and telephone technical support for a low bi-yearly fee._

_The Evolution IVR developer site provides you with VoiceXML and VOIP via FWD and more !_
