[OpenSky](http://www.gizmo5.com/pc/opensky/ "opensky") is a free service provided by Gizmo5 which allows **any** mobile phone, web browser or IP aware phone network (SIP, asterisk, etc) to communicate for up to 5 minutes with Skype users. OpenSky supports sending text messages and voice calls.
