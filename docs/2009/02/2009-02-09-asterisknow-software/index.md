Asterisk® in minutes.

[AsteriskNOW](http://asterisknow.org/ "asterisknow") is an Open Source Software Appliance; a customized GNU/Linux distribution that includes [Asterisk](http://www.asterisk.org/), the AsteriskGUI™ and all other software needed for an Asterisk system.

AsteriskNOW is easy to install, and offers flexibility, functionality and features not available in advanced, high-cost proprietary business systems.
