[FreeSWITCH PBX](http://wiki.freeswitch.org/wiki/Specsheet "freeswitch") features

\* Centralized User/Domain Directory (directory.xml) \* Nano Second CDR granularity \* Call recording (In Stereo caller/callee left/right) \* High Performance Multi-Threaded Core engine \* Configuration via CURL to your http server (xml\_curl). \* XML Config files for easy parsing. \* Protocol Agnostic
