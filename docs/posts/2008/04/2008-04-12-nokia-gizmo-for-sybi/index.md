The [Nokia Gizmo for Sybian S60 BETA](http://www.nokia.com/betalabs/gizmo "nokia") lets you start making internet calls and using instant messaging (IM) and Presence. After installing the application you will be able to create an account or use the existing Gizmo username and password. With a WLAN connection you can use the full set of features - and IM and Presence are available even when you are connected over 2G or 3G.
