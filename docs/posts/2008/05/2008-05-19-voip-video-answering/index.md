The [PSE Video Answering Machine](http://www.mirial.com/products/PSE_Video_Answering_Machi.html "video answering machine") is one of the packaged video applications built on the PSE Media Server available from Mirial DMD. This Video Mail solution, like a telephone answering machine or voicemail system, provides customizable greetings and records incoming caller's messages - all with the added impact of video.
